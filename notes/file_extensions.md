# File Formats and their Applications:

`.md` - Github markdown code

`.rs` - Rust code

`.toml` - Rust Cargo code

`.py` - Python code

`.c/.cpp/.h` - C/C++ code

`.kra` - Krita illustration file

`.csp` - Clip Studio Paint illustration file

`.epgz` - [Pencil](https://pencil.evolus.vn/) flowchart file
